# Choudhary Rafal Singh Educational Trust Website

A modern, responsive website for the Choudhary Rafal Singh Educational Trust, a non-profit organization dedicated to empowering communities through education, healthcare, clean water, and environmental initiatives in Uttarakhand, India.

## 🌟 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Beautiful animations, smooth transitions, and interactive elements
- **Multi-Step Forms**: User-friendly donation and volunteer forms
- **Interactive Timeline**: Shows the trust's journey and milestones
- **Dynamic Components**: 
  - Hero slideshow with particle effects
  - Testimonial carousel
  - Gallery with lightbox
  - Impact calculator
  - Progress bars and counters
- **Contact Integration**: Click-to-call, WhatsApp integration, and Google Maps
- **Accessibility**: Semantic HTML5, ARIA labels, and keyboard navigation

## 🚀 Technologies Used

- **HTML5**: Semantic markup and modern features
- **CSS3**: Custom animations, gradients, and responsive grid layouts
- **JavaScript (Vanilla)**: No dependencies, pure JavaScript for all interactions
- **Font Awesome**: Icons for UI elements
- **Google Fonts**: Typography (Cormorant Garamond, Outfit, Noto Serif Devanagari)

## 📱 Sections

1. **Hero**: Dynamic slideshow with call-to-action
2. **Patron**: Founder's message and vision
3. **About**: Mission and values
4. **Leadership**: Team information with contact details
5. **Our Work**: Four focus areas (Education, Healthcare, Water, Environment)
6. **Impact**: Donation calculator and success metrics
7. **Goals**: Active fundraising campaigns
8. **Testimonials**: Beneficiary stories
9. **Timeline**: Trust's journey and milestones
10. **Gallery**: Photo collection with lightbox
11. **Get Involved**: Multi-step volunteer/donation form
12. **Contact**: Contact information and quick message form

## 🔧 Setup Instructions

1. **Clone or download this repository**
2. **Add your assets**:
   - Place your logo in `assets/logo.png`
   - Add actual photos in the gallery section
   - Update leadership photos
3. **Customize content**:
   - Update contact information
   - Modify mission statements
   - Adjust timeline dates and milestones
4. **Deploy**: Upload to any web hosting service or GitHub Pages

## 🌐 Deployment

### GitHub Pages
1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select source branch (usually `main` or `master`)
4. Your site will be live at `https://username.github.io/repository-name`

### Other Hosting
The website can be deployed on any static hosting service:
- Netlify
- Vercel
- AWS S3
- Traditional web hosting
